# Supabase Setup for ProjectGenie

To make the backend functional, please follow these steps:

## 1. Create a Supabase Project
Go to [Supabase](https://supabase.com/) and create a new project.

## 2. Get Credentials
1. Go to Project Settings -> API.
2. Copy `Project URL` and `anon public` Key.
3. Open `lib/core/services/supabase_service.dart` and paste them into `supabaseUrl` and `supabaseAnonKey`.

## 3. Database Schema

Run the following SQL in your Supabase SQL Editor to create the necessary tables.

### Services Table
```sql
create table services (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  title text not null,
  "desc" text not null,
  icon text not null,
  price text,
  delivery text,
  type text default 'general' -- 'general' for home grid, 'career' for career services
);
```
